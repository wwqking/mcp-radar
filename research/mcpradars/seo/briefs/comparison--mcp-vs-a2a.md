# MCP vs A2A: Agent Protocols Compared (with ACP)

## Direct answer goal
First 80–120 words: MCP (Model Context Protocol) connects *one agent to tools and
data* — it's vertical, model-to-capability. A2A (Agent-to-Agent) connects
*agents to each other* — it's horizontal, agent-to-agent coordination. They
operate at different layers and are often used together: an agent uses MCP to
reach its tools and A2A to delegate to other agents. ACP is a related
agent-communication effort in the same horizontal space. Then link to
/what-is-mcp-server.
